# ROSALIND
Solution scripts for ROSALIND problem sets. Implemented in Python 3.5.

## Overview
Project is organized as follows
```
rosalind/
    problems/
        stronghold/     (Bioinformatics Stronghold Problem Set)
            scripts/    (solution scripts)
                ...
            test/       (test cases)
                in/     (input)
                out/    (output)
```
